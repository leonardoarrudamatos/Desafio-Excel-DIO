# Desafi: Um Organizador de Declaração de Imposto de Renda
O projeto consiste na criação de uma planilha de controle de dados para facilitar a organização de informações necessárias para a declaração de imposto de renda. A ferramenta foi desenvolvida no Excel, utilizando validações de dados, navegação facilitada e funções interativas.
